# Entity Model Specification

## 1. Overview

This document defines the specification for entity models in the Geelato platform.

## 2. File Structure

meta/
`└── {EntityName}/`
    `├── {EntityName}.define.json`    # Entity definition
    `├── {EntityName}.columns.json`   # Column definitions
    `├── {EntityName}.check.json`    # Check constraints
    `├── {EntityName}.fk.json`       # Foreign keys
    `└── {EntityName}.view.sql`      # View definitions
